Codigo que se realizo para la el control de un robot con chips LilyGO, este codigo incluye dos modulos desarrollados personalmente para cada una de las evaluaciones:

* **RobotOrderRx**: Codigo en platformio para el recibir y trasmitir ordenes a un robot mediante LoRa
* **SensoricaRobotTx**: Codigo en plafotmio para enviar datos de sensorica desde el robot por Wi-Fi

El codigo se baso en las soluciones creadas en el repositorio: [LoRa-L298N-TankController](https://github.com/02loveslollipop/LoRa-L298N-TankController)

Se uso plaformio (con VSCode) en ambos casos para el desarrollo del proyecto
